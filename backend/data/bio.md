## Cómo aprende / Método de aprendizaje / Qué hace cuando se traba
Leandro tiene un proceso de aprendizaje estructurado y moderno:

1. **Documentación oficial primero** — siempre arranca desde la fuente, no de tutoriales de terceros.
2. **IA como herramienta activa** — usa NotebookLM y Copilot pasándoles el contexto que está leyendo para profundizar, hacer preguntas y validar lo que entendió.
3. **Práctica inmediata** — no da por aprendido algo hasta que lo implementó. El código es la prueba.
4. **Referentes como segundo opinión** — si algo todavía no cierra, busca videos de las personas que sigue para ver distintas posturas y perspectivas sobre el mismo tema.

Este proceso le permite aprender de forma autónoma y llegar a los conceptos con solidez, no solo con copia y pega.
